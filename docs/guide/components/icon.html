    # Icon 图标

    基于字体的图标集，包含了大多数常见场景的图标

    # 基本使用

    提示

    如果您觉得内置的图标数量不够，或者不合符您的需求，可自行扩展

    通过 <sino-icon> 形式来调用，设置name参数为图标名即可

    <sino-icon name="shoucang"></sino-icon>

    # 修改图标的样式 beta 默认主题

    • 通过color参数修改图标的颜色
    • 通过size参数修改图标的大小，单位为rpx
    <sino-icon name="keji" color="blue" size="28"></sino-icon>

    # API

    # Props

    参数 说明 类型 默认值 可选值
    name 图标名称，见示例图标集，如名称带有/，会被认为是图片图标 String - -
    color 图标颜色 String inherit -
    size 图标字体大小，单位rpx String|Number Inherit -
    index 一个用于区分多个图标的值，点击图标时通过click事件传出 String - -

    # Events

    事件名 说明 回调参数 版本
    click 点击图标时触发 index: 通过props传递的index值 -
