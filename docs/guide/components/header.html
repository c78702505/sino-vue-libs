    # 卡片头

    该组件一般用于卡片头部：

    • 主标题

    • 副标题

    • 显示更多

    • 点击跳转

    # 基本使用

    通过传入title参数确定标题头文字 通过传入more参数来控制是否显示更多

    <sino-header title="课程列表" :more="true"></sino-header>

    # Props

    参数 说明 类型 默认值 可选值
    title 标题 String - -
    subTitle 副标题 String - -
    more 是否显示更多 Boolean false true
